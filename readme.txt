== Description ==

WordPress SEO Check Plugin is a powerful tool designed to help WordPress site administrators and content creators optimize their posts and pages for search engines and user experience. It automatically scans key SEO elements (meta tags, structure, accessibility) and site performance (focusing on Core Web Vitals) and presents actionable insights.

Key Features:

*   **Automated SEO & Performance Audits**: Get instant scores and detailed issues for each post/page.
*   **Editor Integration**: Real-time feedback directly within the Classic Editor meta box and Gutenberg sidebar.
*   **Gemini AI-Powered Meta Tag Suggestions**: Generate optimized titles and descriptions using advanced AI.
*   **Core Web Vitals Monitoring**: Collect and analyze performance metrics from real user data.
*   **Comprehensive Dashboard**: View overall site performance and detailed issue lists.
*   **Manual Full Site Scan**: Trigger a scan for all published posts and pages with a single click.

== Installation ==

1.  Upload the `wp-seo-check` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Navigate to the 'SEO Audit' menu item in your WordPress admin sidebar to view the dashboard.
4.  For AI-powered features, define `WP_SEO_CHECK_GEMINI_API_KEY` in your `wp-config.php` file.

== Frequently Asked Questions ==

= What are Core Web Vitals? =
Core Web Vitals are a set of real-world, user-centered metrics that quantify key aspects of the user experience: Largest Contentful Paint (LCP), First Input Delay (FID), and Cumulative Layout Shift (CLS).

= How do I get a Gemini API Key? =
You can obtain a Gemini API Key from the Google AI Studio. Please refer to the official Google AI documentation for detailed instructions.
